# Network Asset Manager

A professional-grade inventory system that treats infrastructure hardware as Python Objects. This project demonstrates the transition from procedural scripting to Object-Oriented Programming (OOP) for scalable network management.

## 🚀 Key Features
* **Device Modeling:** Uses a `NetworkDevice` class to encapsulate IP addresses, models, and operational states.
* **Automated Connectivity:** Built-in methods to simulate device connections and status updates.
* **Scalable Inventory:** Designed to handle thousands of unique hardware assets with a single shared blueprint.
